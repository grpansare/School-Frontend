Programming Assignment: Build a To-Do List App

Instructions:

  1. Create a new Angular project using the Angular CLI.
  2. Implement a to-do list feature with the following functionalities:
    • Add a new task with a title and description.
    • Display the list of tasks with their titles and descriptions.
    • Mark tasks as completed or remove them.
  3. Apply basic styling to the to-do list for a better user experience.
  4. Implement a service to manage the data for the to-do list.

Submission:

Please submit your completed project by Friday, March 10th, 2023. You can either share the GitHub repository link or submit a compressed file containing your project.
